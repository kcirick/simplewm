# SimpleWM

## Description

  - Minimal stacking window manager
  - Written in C++
  - Features:
   - No frills (menu, titlebar, icons, pixmap themes, autostart etc...)
   - Text config file
   - ~~Simple (one-tier) root menu~~
   - Window grouping
